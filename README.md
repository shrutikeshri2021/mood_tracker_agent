# ZenithMe - AI-Powered Mood Tracker & Wellness Platform

ZenithMe is a premium, mobile-first wellness application designed to help users track their mental health, predict burnout risks using AI, and access clinical-grade healing tools. Built with a stunning glassmorphism UI, it provides a private and interactive experience for personal growth and emotional well-being.

## 🔗 Repository & Live Demo
- **GitHub Repository:** [https://github.com/shrutikeshri2021/mood_tracker_agent](https://github.com/shrutikeshri2021/mood_tracker_agent)
- **Live Site:** [https://mood-tracker-agent.onrender.com](https://mood-tracker-agent.onrender.com)

## ✨ Key Features
- **Mood Logging & Tracking:** Log your daily emotions with ease.
- **AI Analytics & Forecasting:** Predictive engine for burnout risk and emotional trends.
- **Advanced Healing Tools:** Clinical-grade features for emotional recovery.
- **Doctor-Prescribed Workflows:** Features built for clinical alignment.
- **Privacy First:** Data is handled with privacy in mind.
- **Stunning UI:** Modern, responsive design with smooth animations and glassmorphism.

## 🛠️ Tech Stack
- **Frontend:** React 19, Vite
- **Styling:** Tailwind CSS 4.0, Framer Motion
- **Data Visualization:** Recharts
- **Icons:** Lucide React
- **Deployment:** Render

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/shrutikeshri2021/mood_tracker_agent.git
   ```
2. Navigate to the project directory:
   ```bash
   cd mood_tracker_agent
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Deployment
The project is configured for deployment on **Render** via `render.yaml`. It is automatically built and deployed from the `main` branch.
